Introduction à Julia

Julia est un langage de programmation performant, conçu pour le calcul scientifique et l'analyse de données. Il combine une syntaxe claire similaire à Python avec des performances proches de C/C++. Julia est de plus en plus populaire pour le calcul intensif et l'analyse de données.

Cours : Types de Variables en Julia

En programmation, un type de variable détermine la nature des données qu'elle contient.

  • Int : pour les entiers (ex : a = 5).
  • Float64 : pour les réels à précision 64 bits (ex : b = 3.14).
  • String : pour les chaînes de caractères (ex : c = "Bonjour").
  • Bool : pour les valeurs booléennes vrai/faux (ex : d = true).

Cours : Opérateurs et Expressions en Julia

Les opérateurs sont des symboles utilisés pour effectuer des opérations sur des valeurs. Voici les différents types d'opérateurs en Julia :

1. Types d'Opérateurs en Julia

a) Opérateurs Arithmétiques

Les opérateurs arithmétiques permettent de réaliser des calculs de base.

Opérateur Description Exemple
+ Addition 5 + 3 renvoie 8
- Soustraction 5 - 3 renvoie 2
* Multiplication 5 * 3 renvoie 15
/ Division (résultat réel) 5 / 2 renvoie 2.5
÷ Division entière 5 ÷ 2 renvoie 2
% Modulo (reste de division) 5 % 2 renvoie 1
^ Puissance 2 ^ 3 renvoie 8
b) Opérateurs de Comparaison

Les opérateurs de comparaison permettent de comparer des valeurs. Ils renvoient toujours un booléen (true ou false).

Opérateur Description Exemple
== Égalité 5 == 5 renvoie true
!= Différent 5 != 3 renvoie true
> Supérieur 5 > 3 renvoie true
< Inférieur 5 < 3 renvoie false
>= Supérieur ou égal 5 >= 5 renvoie true
<= Inférieur ou égal 5 <= 3 renvoie false
c) Opérateurs Logiques

Les opérateurs logiques permettent de combiner des expressions booléennes.

Opérateur Description Exemple
&& ET logique (5 > 3) && (3 > 1) renvoie true
! NON logique (négation) !(5 > 3) renvoie false

2. Exemples d'Expressions en Julia

Exemple 1 : Utilisation des opérateurs arithmétiques

a = 10
b = 3

somme = a + b           # Résultat : 13
difference = a - b      # Résultat : 7
produit = a * b         # Résultat : 30
division = a / b        # Résultat : 3.333...
division_entiere = a ÷ b # Résultat : 3
reste = a % b           # Résultat : 1
puissance = a ^ b       # Résultat : 1000 (10 à la puissance 3)
Exemple 2 : Comparaison de valeurs

x = 7
y = 5

println(x == y)         # Résultat : false
println(x != y)         # Résultat : true
println(x > y)          # Résultat : true
println(x <= y)         # Résultat : false
Exemple 3 : Opérateurs logiques

c = true
d = false

println(c && d)         # Résultat : false
println(c || d)         # Résultat : true
println(!c)             # Résultat : false
println(!(c && d))      # Résultat : true

Cours : Conditions et Boucles en Julia


            

                

                    
Structure Conditionnelle if...else

                    
Permet d'exécuter du code en fonction d'une condition.

                    
if condition
    # code si condition est vraie
elseif autre_condition
    # code si autre_condition est vraie
else
    # code si toutes les conditions sont fausses
end
                    


                    
Boucles en Julia

                    
Boucle for : Répète un bloc de code pour chaque valeur d'une collection ou d'une plage de valeurs.

                    
for i in 1:5
    println("Itération $i")
end
                    


                    
Boucle while : Continue d'exécuter un bloc tant qu'une condition est vraie.

                    
compteur = 1
while compteur <= 5
    println("Compteur = $compteur")
    compteur += 1
end

Les Fonctions en Julia

        
            

                

                    
1. Définition de Fonction en Julia

                    
En Julia, une fonction se définit à l'aide du mot-clé function, suivi d'un nom de fonction, d'une liste d'arguments entre parenthèses, et du corps de la fonction. La fonction se termine par end.

        
                    
function nom_de_fonction(argument1, argument2, ...)
            # Corps de la fonction
            return resultat
        end

        
                    
    
                        
  • nom_de_fonction : le nom donné à la fonction pour l’appeler.
    • 
                        
  • arguments : valeurs d'entrée nécessaires pour le traitement de la fonction.
    • 
                        
  • return : mot-clé permettant de renvoyer un résultat. Si omis, la fonction retourne la dernière expression évaluée.
    • 
                    

        
                    
Exemple de Fonction Simple

                    
function addition(a, b)
            return a + b
        end
        
        resultat = addition(3, 5)  # Résultat : 8
        println(resultat)

                    
Dans cet exemple, la fonction addition prend deux paramètres a et b, calcule leur somme et renvoie le résultat.

                

            

        
            

                

                    
2. Fonctions sans Arguments

                    
Les fonctions en Julia peuvent également ne prendre aucun argument. Elles réalisent alors des tâches prédéfinies sans nécessiter de données d'entrée.

        
                    
function saluer()
            println("Bonjour !")
        end
        
        saluer()  # Affiche : Bonjour !

                

            

        
            

                

                    
3. Valeurs de Retour

                    
Le mot-clé return est optionnel. En son absence, Julia retourne la dernière expression calculée.

        
                    
function multiplier_par_deux(x)
            x * 2  # Retour implicite de cette expression
        end
        
        println(multiplier_par_deux(5))  # Résultat : 10

                

            

        
            

                

                    
4. Fonctions avec Valeurs Par Défaut

                    
Il est possible de définir des valeurs par défaut pour les arguments d'une fonction. En cas d'absence de valeur lors de l'appel, Julia utilise la valeur par défaut.

        
                    
function bonjour(nom="anonyme")
            println("Bonjour, $nom!")
        end
        
        bonjour("Alice")  # Affiche : Bonjour, Alice!
        bonjour()         # Affiche : Bonjour, anonyme!

                

            

        
            

                

                    
5. Fonctions Anonymes

                    
Les fonctions anonymes n'ont pas de nom et sont souvent utilisées pour des tâches simples. La syntaxe est arguments -> expression.

        
                    
addition = (a, b) -> a + b
        println(addition(3, 7))  # Résultat : 10

                

            

        
            
Exemples Pratiques

        
            

                

                    
Exemple 1 : Fonction de Calcul de l'Aire d'un Rectangle

                    
function aire_rectangle(longueur, largeur)
            return longueur * largeur
        end
        
        aire = aire_rectangle(5, 3)  # Résultat : 15
        println("L'aire est : $aire")

                

            

        
            

                

                    
Exemple 2 : Fonction de Salutation avec Valeur Par Défaut

                    
function saluer(nom="Inconnu")
            println("Bonjour, $nom!")
        end
        
        saluer("Marie")  # Affiche : Bonjour, Marie!
        saluer()         # Affiche : Bonjour, Inconnu!

                

            

        
            

                

                    
Exemple 3 : Fonction pour Vérifier la Parité d'un Nombre

                    
function est_pair(x)
            return x % 2 == 0
        end
        
        println(est_pair(4))  # Résultat : true
        println(est_pair(7))  # Résultat : false
